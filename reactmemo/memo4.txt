lazy関数は、コンポーネントを使うときに初めて読み込むための仕組み
普通のimportはページを開いた瞬間読み込まれるが、lazyは初期表示時には読み込まれない
スクロールした瞬間に読み込んでくれる画像、とか
遅延ロード(初期表示を軽くする)とSuspense(読み込めてない時別の表示をさせるもの)は別物
import { lazy, Suspense } from 'react';

const UserProfile = lazy(() => import('./UserProfile'));

function App() {
  return (
    <Suspense fallback={<div>読み込み中...</div>}>
      <UserProfile />
      ここだけ何秒後かに表示される
    </Suspense>
  );
}

lazyはこう書く、基本的に何かをimportしたいときに使う
cons LazyButton = lazy(() => import('./Lazy'))
