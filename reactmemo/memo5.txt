---Reactライブラリ(誰かが作ってくれた便利な機能の集まり)の活用---
MUI(マテリアルUI) いろんなwebサイトで使われている googleが作った
(Reactを作ったのはmeta)
ライブラリとして作られたコンポーネントをいっぱい提供してくれるライブラリ
MUIはそのReact上で使う、既製品の部品セット


<MUIの例>
import Button from '@mui/material/Button';
// MUIのボタン(最初から綺麗にデザインされている)
<Button variant="contained">送信</Button>
誰かが作ってくれたデザインを活用すればよくねっていう

ライブラリなのでインストールする必要がある
pnpm add @mui/material @mui/icons-material @emotion/react @emotion/styled

・ボタン
primary => 青
secondary => 紫
sx={{ textTransform: 'none' }} 書いたまま表示してくれる

・ドロワーメニュー

・Grid...画面を12等分して部品を並べるためのレイアウト
containerをつけることで「このGridは子要素を並べるための入れ物」
spacingは子要素同士の隙間
fullWidthはボタンの大きさを決める
(fullWidthがないとボタンがそのスペースだけ大きくはならない
スペースはその大きさだけ使うが、=ボタンがその大きさというわけではない)
spacingは1=8px

・レスポンシブルレイアウト
タブレットやスマホなどどの環境でもレイアウトを崩さずに表示できるデザインのこと
xs → スマホ📱
sm → タブレット📱
md → ノートPC💻
lg → デスクトップ🖥
xl → 大きなモニター🖥

・自作テーマをアプリに適用
Themeでスタイルに一貫性を持たせられる、オレンジって指定したら基本オレンジになる？
paletteでprimary,secondaryを囲って、一括で色を変えることができる[500]は色の濃さ
spacing:で基準を変える
main.tsxにthemeをimportさせて自作テーマに一貫性を持たせられる
